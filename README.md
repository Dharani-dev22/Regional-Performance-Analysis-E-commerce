# \# E-Commerce Sales Performance Analysis

# 

# \## 🚀 Project Overview

# This project demonstrates an end-to-end data analytics pipeline, transforming raw e-commerce transaction data into actionable business insights. The focus was on building a robust workflow—from database architecture to final dashboard visualization.

# 

# \## 🛠️ Technical Stack

# \* \*\*SQL (SQLite):\*\* Used for data extraction, joining tables, and calculating revenue metrics.

# \* \*\*Excel:\*\* Performed data validation and calculated key performance indicators (KPIs) like Average Order Value (AOV).

# \* \*\*Tableau Public:\*\* Developed an interactive dashboard to visualize regional performance and trends.

# \* \*\*Git/GitHub:\*\* Managed version control and project documentation.

# 

# \## 📈 Key Business Insights

# \- \*\*Regional Performance:\*\* Mumbai is the top-performing region, contributing the highest volume of total revenue.

# \- \*\*Consumer Behavior:\*\* Data suggests a higher propensity for premium purchases in Mumbai compared to Delhi, driven by a higher Average Order Value.

# \- \*\*Data Pipeline:\*\* Successfully established a repeatable workflow for cleaning, validating, and visualizing sales data.
