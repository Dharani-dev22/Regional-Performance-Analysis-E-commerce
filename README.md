# E-Commerce Sales Performance Analysis

## 🚀 Project Overview

This project demonstrates an end-to-end data analytics pipeline, transforming raw e-commerce transaction data into actionable business insights. The focus was on building a robust workflow—from database architecture to final dashboard visualization.

## 🛠️ Technical Stack

* **SQL (SQLite):** Data extraction, joining tables, and calculating revenue metrics.
* **Excel:** Data validation and calculated KPIs like Average Order Value (AOV).
* **Tableau Public:** Developed interactive dashboards to visualize regional performance.
* **Git/GitHub:** Version control and documentation.

## 📈 Key Business Insights

* **Regional Performance:** Mumbai is the top-performing region by total revenue.
* **Consumer Behavior:** Mumbai shows a higher propensity for premium purchases compared to Delhi.
* **Data Pipeline:** Successfully established a repeatable workflow for cleaning, validating, and visualizing sales data.
